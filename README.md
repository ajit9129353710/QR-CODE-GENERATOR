📱 QR Code Generator

A simple QR Code Generator built using HTML, CSS, and JavaScript.

You can enter any text or website URL and generate a QR code instantly.

✨ Features

- ⭐ Generate QR code from text or URL
- ⭐ Simple and clean UI
- ⭐ Responsive design
- ⭐ Fast and easy to use

🛠️ Technologies Used

- HTML
- CSS
- JavaScript

📂 Project Files

- 📄 "index.html"
- 🎨 "style.css"
- ⚡ "script.js"

🚀 How to Use

1. Open "index.html" in your browser.
2. Enter any text or URL.
3. Click the Generate QR button.
4. Your QR code will be displayed instantly.
